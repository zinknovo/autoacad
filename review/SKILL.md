# AutoAcad Review

## Overview

Use this stage for adversarial review. Treat unsupported claims as critical failures, not editing suggestions.

## Review Focus

1. Topic discipline: does every section connect back to the core research problem?
2. Method-evidence consistency: do claimed trials, tests, metrics, and baselines match the actual code and results?
3. Baseline strength: were comparators tuned fairly?
4. Ablation coverage: does each claimed component have an ablation?
5. Length and depth: are all major sections substantial enough for a top-tier submission?
6. Reviewer defense: what is the harshest plausible objection and how should the paper answer it?

## Output

Produce a reviewer-style report with strengths, weaknesses, critical fabrications if any, and concrete revision actions.
