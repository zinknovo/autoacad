# Canonical Paper Project Tree

```text
<project>-paper/
├── MEGA_PROMPT.md
├── RESTRICTS.yaml
├── PROGRESS.md
├── code/
├── data/
├── docs/
├── paper/
│   ├── <conference-template>/
│   └── mypaper/
│       ├── figures/
│       ├── main.tex
│       └── sections/
├── plans/
└── results/
```

Notes:
- `PROGRESS.md` is mandatory.
- Put per-stage plans in `plans/`.
- Save executable experiment code in `code/` and machine-readable outputs in `results/`.
