# AutoAcad Paper Structure

## Target Structure

| Section | Target Words | Purpose |
| --- | --- | --- |
| Abstract | 150-250 | Problem, method, key numeric result, conclusion. |
| Introduction | 800-1000 | Motivation, gap, method summary, contributions, paper map. |
| Related Work | 600-800 | Thematic comparison and positioning. |
| Method | 1000-1500 | Problem definition, algorithm, rationale, complexity. |
| Experiments | 800-1200 | Setup, datasets, baselines, metrics, hardware, hyperparameters. |
| Results | 600-800 | Main tables, ablations, statistical discussion, qualitative findings. |
| Discussion | 400-600 | Interpretation, implications, unexpected outcomes. |
| Limitations | 200-300 | Scope, data, and generalization limits. |
| Conclusion | 200-300 | Final contribution summary and future work. |

## Core Paper Rules

1. Figure 1 must be planned before full drafting.
2. Every effective component mentioned in the paper needs an ablation.
3. Baselines must receive comparable tuning effort.
4. Citations must support claims, not decorate them.
5. Keep the main body near the page budget without exceeding it.

## Drafting Order

Recommended order: outline -> figure plan -> method/experiments -> results/discussion -> introduction/related work -> abstract -> revision.
