# AutoAcad Draft

## Overview

Use this stage to turn evidence into a submission-grade manuscript.

## Workflow

1. Build a detailed outline before drafting. Map each section to concrete evidence.
2. Lock Figure 1 early. The first figure must communicate the paper's core idea independently.
3. Draft section by section with real numbers from the result artifacts.
4. Respect section depth targets from `../references/paper-structure.md`.
5. Keep the paper focused on 1-2 core ideas. Drop side quests that do not strengthen acceptance odds.
6. If content does not fit page budget, move peripheral detail to appendix instead of weakening the core narrative.

## Non-Negotiables

- No fabricated metrics.
- No claims without either citations or experiment support.
- Limitations must be explicit.
